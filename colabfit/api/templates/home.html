ColabFit Data Repository

The ColabFit Data Repository is a collection of datasets intended for training, testing, and validating data-driven interatomic potentials. The data in this repository was curated by the ColabFit group at the University of Minnesota, and is drawn from the literature and user submissions.

This website is currently styled after the free "Small Business" Bootstrap template.
